jQuery.Viewport

Fork me on GitHub

Very helpfull, simple and lightweight jQuery plugin that makes an element as a viewport for displaying elements with absolute position in it. Inspired by working on HTML5 puzzle game. You can see it in action right there:

There are two additional plugins are used in this example: jquery.draggable (for dragging) from the jQuery UI library and my another fancy-pancy plugin named jquery.scraggable (for scrolling).

Code example

CSS
#element {
    height:600px;
    width:350px;
}
HTML
<div id="element">
    <img src="http://borbit.org/img/dragme.gif" id="image" />
</div>
JS
$(function() {
    $('#element').viewport();

    var content = $('#element').viewport('content');

    content.draggable({containment: 'parent'});
    content.scraggable({containment: 'parent'});
});

Documentation

Options

{content: false}

Reference or selector to the DOM element which will be shown as content in the viewport box. Possible values: HTMLElement, false, jQueryElement, '.selector'. In case if this option is not set or it is false all children elements the viewport element contains will be taken as a content.

{contentClass: 'viewportContent'} Content element's class name.
{binderClass: 'viewportBinder'} Binder element's class name.

Methods

'update'

Updates binder's position and size and updates content position regarding the viewport element's size. This method should be called if viewport element's size is changed. Return value jQueryElement.

'size'

Sets or returns content element's size. It is useful when your content doesn't have static width and height. Return value: if parameters are not set - {height: [number], width: [number]}, else - jQueryElement.

'content' Returns a jQuery object with content's element.
'binder' Returns a jQuery object with binder's element.
